# Term Project Frontend (Vite + React)

## Project Structure

```
term_projectFE/
├─ public/
│  └─ images/                # public 정적 이미지
├─ src/
│  ├─ components/            # 공용 컴포넌트
│  │  └─ authContext.jsx     # 인증 Context
│  ├─ pages/                 # 페이지 단위 컴포넌트
│  ├─ images/                # (필요 시 import용 이미지)
│  ├─ styles/                # CSS 파일
│  ├─ App.jsx
│  └─ main.jsx               # Vite 엔트리 포인트
├─ index.html                # Vite 엔트리 HTML
├─ vite.config.js
└─ package.json
```

## Getting Started

### Install

```
npm install
```

### Run Dev Server

```
npm run dev
```

기본 주소: http://localhost:5173
